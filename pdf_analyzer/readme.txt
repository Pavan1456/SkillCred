# Pdf Analyzer Project

## Description
This project is designed to analyze PDF files and extract meaningful data. This tool can be particularly useful for processing and extracting information from documents such as resumes, certificates, and reports. It can answer your question from pdf. 

## Features
- Extract text from PDF files
- Process and analyze the extracted data
- Customizable to handle various types of PDFs

## Installation
To run this project, you will need Python installed on your machine. 

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/chat-bot.git
2. Navigate to the project directory:

         cd pdf_analyzer
3. Create a virtual environment:


         python -m venv venv
4. Activate the virtual environment:

   On Windows:
         venv\Scripts\activate
   On macOS/Linux:

         source venv/bin/activate
5. Install the required dependencies:

         pip install -r require.txt

6. Usage

   IN Terminal
         
         streamlit run pdf_analyzer.py
